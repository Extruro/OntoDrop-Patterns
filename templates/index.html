OntoDrop Patterns

OntoDrop identify and extract patterns from a collection of ontologies.

Steps to follow

  1. Prepare the ontology or set of ontologies you want to transform.
  2. They must be inside a .zip file or in .csv if you want to download them.
  3. Attach only one file with the ontology(ies) by dragging and dropping or clicking on the service box.
  4. Once attached, the file name appears in the box. You can now click on "Submit".
  5. Wait until "SUCCESSFULLY COMPLETED" appears.
  6. Now you can download the patterns by clicking on the "Download Pattern" button.
  7. A .zip file with the results will be downloaded.
  8. If you need more information about using the tool, click here.

1) Choose options

Patterns

Is a flag to indicate if the patterns are going to be created from the type of the terms or from the name of the terms or from both. By default "type" is selected.

Flatten

Is a flag to indicate if the collections are going to be flattened if they only contain named classes. By default "no" is selected.
