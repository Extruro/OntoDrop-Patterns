Prepare files

The first step is to have the ontologies ready in a zip file if we have them locally or in a csv file with the name and URI in case we want to download them.

Attach the file

Once the ontology file is ready, the next step is to attach the file to the service box. This can be done in two ways. The first is by dragging and dropping into the box and the second is by clicking inside the box and choosing the file.

Choose options

When you attach the file, you will get a window to choose two optional parameters depending on what you want to do.

-Patterns: is a flag to indicate if the patterns are going to be created from the type of the terms or from the name of the terms or from both.

-Flatten: is a flag to indicate if the collections are going to be flattened if they only contain named classes.

Transforming ontology

Once the options have been chosen, click on the Submit button to start the transformation. The next step is to wait until "SUCCESSFULLY COMPLETED" appears and our zip result will be ready to download it by clicking on the "Download Pattern" button.

Results

When you click on the "Download Pattern" button, you will download "patterns.zip" containing the results with the patterns.
