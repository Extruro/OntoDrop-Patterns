1.Choose Options

Here you can choose your options. Both fields must be filled in now or later if you want to send your files. Each parameter has an information icon with examples of each output.


If we click on the information icon we will find examples for each of the outputs.


1.1 Patterns Info

-Type: the patterns are going to be created from the type of the terms.

-Name: the patterns are going to be created from the name of the terms.

-Both: created from both.







1.2 Flatten Info

-Yes: the collections are going to be flattened.

-No: the collections are not going to be flattened.

2.Prepare files

The first step is to have the ontologies ready in a zip file if we have them locally or in a csv file with the name and URI in case we want to download them.

3.Attach the file

Once the ontology file is ready, the next step is to attach the file to the service box. This can be done in two ways. The first is by dragging and dropping it into the box and the second is by clicking inside the box and choosing the file. In case the two parameters have not been selected, a window will pop up where you can do it.

4.Generating patterns

Once the options have been chosen, click on the Submit button to start the transformation. The next step is to wait until "SUCCESSFULLY COMPLETED" appears and our zip result will be ready to download it by clicking on the "Download Pattern" button.

5.Results

When you click on the "Download Pattern" button, you will download "patterns.zip" containing the results with the patterns.
